# movie-site-research-archive
本仓库用于开展影视类网站的信息研究与资料归档工作，主要整理互联网上已公开存在的电影、电视剧相关站点信息。记录内容包括站点访问地址、页面类型、内容分类方向及基础结构特征，便于进行长期观察与横向对比。本项目不涉及任何影视内容的播放、存储或传播，仅用于信息层面的整理与研究分析。所有资料均来源于公开网络环境，不构成对所记录站点的推荐、评价或商业背书。
